🚀MagicLitePlanet - 适用于小型团体的协作系统

  交流、工作、活动、分析、管理、审批...等的功能均不可思议地集成于优雅而简约轻便的 **WePlanet(Desktop)**。

- 平台：Windows x64
- 开发环境：Visual Studio 2022 MSVC
- 类型：Qt桌面应用程序
- 数据库：MySQL5.7
- 语言：C++ 11
- 框架：Qt5.15.x
